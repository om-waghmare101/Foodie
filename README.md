## рџ“– How It Works

1. Browse available food items.
2. Use the search bar to filter food items.
3. Click **Add to Cart** to add products.
4. Open the cart from the navigation bar.
5. Increase or decrease quantities.
6. Remove unwanted items.
7. View the total amount.
8. Click **Place Order**.
9. Scan the QR code to complete payment.
