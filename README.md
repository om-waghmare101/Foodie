# 🍔 Foodie - Food Ordering Web Application

Foodie is a responsive food ordering web application developed using **HTML, CSS, JavaScript, and React.js**. The application allows users to browse food items, search for dishes, add items to the cart, manage quantities, and place orders with a QR code payment option.

---

## 📸 Preview

### Home Page
- Responsive navigation bar
- Food cards with images, prices, and Add to Cart button
- Search functionality

### Cart Sidebar
- View selected items
- Increase/Decrease quantity
- Remove items
- Dynamic total calculation
- Place Order button
- QR Code payment option

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/foodie.git
```

### Navigate to the project

```bash
cd foodie
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```

*(If using Create React App, use `npm start` instead.)*

## 📖 How It Works

1. Browse available food items.
2. Use the search bar to filter food items.
3. Click **Add to Cart** to add products.
4. Open the cart from the navigation bar.
5. Increase or decrease quantities.
6. Remove unwanted items.
7. View the total amount.
8. Click **Place Order**.
9. Scan the QR code to complete payment.
